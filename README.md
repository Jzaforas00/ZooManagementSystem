# Zoo Management System 🦁

Zoo Management System built with ASP.NET Core MVC and .NET 8.

## 📌 Description

This project simulates the management of a zoo, including animals, employees, feeding control, and ecosystem organization.

It has been developed as part of a learning process to practice clean architecture, MVC pattern, and database management using Entity Framework.

## 🛠️ Technologies

- ASP.NET Core MVC (.NET 8)
- C#
- Entity Framework Core
- SQL Server (Local DB)
- HTML, CSS

## 📁 Project Structure

- Controllers → Application logic
- Models → Entities and ViewModels
- Data → DbContext and repositories
- Services → Business logic
- Views → UI (Razor)
- ViewComponents → Reusable UI components

## ✨ Features

- Animal management (CRUD)
- Feeding and diet control
- Ecosystem and cage organization
- Worker roles (Admin, Feeding, Veterinary)
- Basic frontend for visitors to view animals 

## 📚 Learning Purpose

This project was developed to strengthen and apply the following concepts:

- ASP.NET Core MVC architecture
- Entity Framework Core and database relationships
- Separation of concerns (Controllers, Services, Data layers)
- Clean code and maintainable project structure
- Working with relational databases and data modeling
